## 接口说明

要求请求者每 10 秒发送一次心跳。30 秒内如果没有收到心跳请求，服务端将断开 WebSocket 连接。

## 请求 - 协议号：22000

```json
{
  "cmd_id": 22000,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```

## 应答 - 协议号：22001

```json
{
  "ret": 200,
  "msg": "ok",
  "cmd_id": 22001,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```
