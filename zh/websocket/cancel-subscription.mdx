## 接口说明

取消已订阅的报价推送。

## 请求 - 协议号：22006

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| cancel_type | uint32 | 是 | 0=取消所有，1=取消盘口，2=取消成交，3=取消汇率 |

```json
{
  "cmd_id": 22006,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {
    "cancel_type": 1
  }
}
```
