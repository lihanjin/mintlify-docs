## 接口说明

该接口支持订阅产品的最新盘口（实时逐笔深度、Order Book）数据。每发送一次订阅请求，后台会覆盖上一次订阅。

各产品最大盘口深度：

| 产品类型 | 最大深度 |
|----------|----------|
| 外汇、贵金属、原油 | 1 档 |
| 加密货币 | 5 档 |
| 港股 | 10 档 |
| 美股 | 1 档 |
| 沪深A股 | 5 档 |

## 请求 - 协议号：22002

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| cmd_id | integer | 是 | 固定值：22002 |
| seq_id | integer | 是 | 订阅请求标识 |
| trace | string | 是 | 请求追溯 ID |
| data.symbol_list | array | 是 | 产品列表 |
| data.symbol_list[].code | string | 是 | 产品代码 |
| data.symbol_list[].depth_level | integer | 否 | 深度层级（不传则返回1层） |

```json
{
  "cmd_id": 22002,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {
    "symbol_list": [
      { "code": "BTCUSDT", "depth_level": 5 },
      { "code": "ETHUSDT", "depth_level": 5 }
    ]
  }
}
```

## 应答 - 协议号：22003

```json
{
  "ret": 200,
  "msg": "ok",
  "cmd_id": 22003,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```

## 推送 - 协议号：22999

| 字段 | 类型 | 说明 |
|------|------|------|
| code | string | 产品代码 |
| seq | string | 报价序号 |
| tick_time | string | 报价时间戳（毫秒） |
| bids | array | 买盘列表 [{price, volume}] |
| asks | array | 卖盘列表 [{price, volume}] |

```json
{
  "cmd_id": 22999,
  "data": {
    "code": "BTCUSDT",
    "seq": "1605509068000001",
    "tick_time": "1605509068",
    "bids": [
      { "price": "9.12", "volume": "100" }
    ],
    "asks": [
      { "price": "9.13", "volume": "150" }
    ]
  }
}
```
