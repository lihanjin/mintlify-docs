## HTTP 请求通用标准头

| 字段 | 名称 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| trace | 跟踪号 | string | 是 | 请求者生成唯一，响应与请求保持一致，最大长度64 |
| data | 数据体 | object | 是 | 具体数据格式见各个接口定义 |

请求示例：

```json
{
  "trace": "c2a8a146-a647-4d6f-ac07-8c4805bf0b74",
  "data": {}
}
```

## HTTP 应答通用标准头

| 字段 | 名称 | 类型 | 说明 |
|------|------|------|------|
| ret | 返回值 | int32 | 错误码 |
| msg | 消息 | string | 对成功或失败的具体描述 |
| trace | 跟踪号 | string | 与请求保持一致 |
| data | 数据体 | object | 具体数据格式见各个接口定义 |

应答示例：

```json
{
  "ret": 200,
  "msg": "ok",
  "trace": "c2a8a146-a647-4d6f-ac07-8c4805bf0b74",
  "data": {}
}
```

## WebSocket 请求通用格式

| 字段 | 名称 | 类型 | 必填 | 说明 |
|------|------|------|------|------|
| cmd_id | 协议号 | integer | 是 | 接口协议号 |
| seq_id | 响应id | integer | 是 | 订阅请求标识，响应回传 |
| trace | 可追溯id | string | 是 | 请求日志信息可追溯id |
| data | 数据体 | object | 是 | 具体数据格式见各个接口定义 |

## 错误码说明

| 错误码 | 说明 |
|--------|------|
| 200 | 成功 |
| 202 | 请求数据参数无效 |
