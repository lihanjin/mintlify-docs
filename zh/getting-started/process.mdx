本指南带你完成从注册到成功调用 AllTick API 的全过程。最终你将获得一个可运行的代码示例，使用自己的 Token 查询 AAPL.US 的日 K 线数据。

## 1. 注册账号

前往 [alltick.co/register](https://alltick.co/register) 完成注册：

1. 输入有效邮箱地址
2. 设置密码（需包含大小写字母、数字和特殊字符）
3. 输入邮箱收到的验证码
4. 点击注册，成功后自动跳转到仪表盘

## 2. 获取 API Token

进入仪表盘后，导航至 **API Keys** 部分，复制你的 Token。

<Warning>
Token 是你的唯一凭证，请妥善保管。如果重新生成 Token，旧 Token 立即失效。
</Warning>

## 3. 选择 API 端点

AllTick 按资产类别使用不同的 Base URL：

| 资产类别 | Base URL |
|----------|----------|
| 美股、港股、A 股、指数 | `https://quote.alltick.co/quote-stock-b-api` |
| 外汇、加密货币、贵金属、商品 | `https://quote.alltick.co/quote-b-api` |

## 4. 发起第一个请求

以查询 AAPL.US 最近 10 根日 K 线为例：

<Card title="在 API Playground 中试一试" icon="play" href="/zh/api-reference/stock/kline">
  打开 K 线接口的交互式 Playground，填入 Token 即可直接发起请求
</Card>

代码示例：

<CodeGroup>

```python Python
import requests
import json
import urllib.parse

token = "YOUR_TOKEN"
query = json.dumps({
    "trace": "test-001",
    "data": {
        "code": "AAPL.US",
        "kline_type": 8,
        "kline_timestamp_end": 0,
        "query_kline_num": 10,
        "adjust_type": 0
    }
})

url = f"https://quote.alltick.co/quote-stock-b-api/kline?token={token}&query={urllib.parse.quote(query)}"
response = requests.get(url)
print(json.dumps(response.json(), indent=2))
```

```go Go
package main

import (
    "fmt"
    "io"
    "net/http"
    "net/url"
)

func main() {
    token := "YOUR_TOKEN"
    query := `{"trace":"test-001","data":{"code":"AAPL.US","kline_type":8,"kline_timestamp_end":0,"query_kline_num":10,"adjust_type":0}}`

    reqURL := fmt.Sprintf("https://quote.alltick.co/quote-stock-b-api/kline?token=%s&query=%s",
        token, url.QueryEscape(query))

    resp, err := http.Get(reqURL)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()
    body, _ := io.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```

</CodeGroup>

将 `YOUR_TOKEN` 替换为你在仪表盘获取的实际 Token。
