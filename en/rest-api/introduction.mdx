## Overview

AllTick REST API provides the following HTTP endpoints:

| Endpoint | Method | Description |
|----------|--------|-------------|
| /kline | GET | Single product historical K-line query |
| /batch-kline | POST | Batch query latest 2 K-lines for multiple products |
| /depth-tick | GET | Latest order book (market depth) query |
| /trade-tick | GET | Latest trade price batch query |

## Base URLs

**Stocks (US, HK, A-shares):**
- `https://quote.alltick.co/quote-stock-b-api`

**Forex, Crypto, Precious Metals, Commodities:**
- `https://quote.alltick.co/quote-b-api`

## Request Format

The `query` parameter for GET requests must be a URL-encoded JSON string:

```bash
# Original JSON
{
  "trace": "unique-trace-id",
  "data": {
    "code": "700.HK",
    "kline_type": 1,
    "kline_timestamp_end": 0,
    "query_kline_num": 2,
    "adjust_type": 0
  }
}

# URL-encode and append to query parameter
https://quote.alltick.co/quote-stock-b-api/kline?token=YOUR_TOKEN&query=%7B%22trace%22...%7D
```

## K-Line Types

| Value | Type |
|-------|------|
| 1 | 1 minute |
| 2 | 5 minutes |
| 3 | 15 minutes |
| 4 | 30 minutes |
| 5 | 1 hour |
| 6 | 2 hours (stocks not supported) |
| 7 | 4 hours (stocks not supported) |
| 8 | Daily |
| 9 | Weekly |
| 10 | Monthly |
