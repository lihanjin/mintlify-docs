## Description

Subscribe to latest trade prices (real-time tick data) for multiple products. Each subscription request overwrites the previous one.

<Warning>
To add products to an existing subscription, resend the complete product list. For example, if subscribed to A, B, C and want to add D, E, send A, B, C, D, E.
</Warning>

## Request - Command ID: 22004

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| cmd_id | integer | Yes | Fixed: 22004 |
| seq_id | integer | Yes | Request sequence ID |
| trace | string | Yes | Trace ID (unique per request) |
| data.symbol_list | array | Yes | Product list |
| data.symbol_list[].code | string | Yes | Product code |

```json
{
  "cmd_id": 22004,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {
    "symbol_list": [
      { "code": "BTCUSDT" },
      { "code": "ETHUSDT" }
    ]
  }
}
```

## Response - Command ID: 22005

```json
{
  "ret": 200,
  "msg": "ok",
  "cmd_id": 22005,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```

## Push - Command ID: 22998

| Field | Type | Description |
|-------|------|-------------|
| code | string | Product code |
| seq | string | Quote sequence number |
| tick_time | string | Timestamp (milliseconds) |
| price | string | Latest trade price |
| volume | string | Trade volume |
| turnover | string | Trade turnover |
| trade_direction | integer | Direction: 0=neutral, 1=buy, 2=sell |

```json
{
  "cmd_id": 22998,
  "data": {
    "code": "BTCUSDT",
    "seq": "1605509068000001",
    "tick_time": "1605509068",
    "price": "651.12",
    "volume": "300",
    "turnover": "12345.6",
    "trade_direction": 1
  }
}
```
