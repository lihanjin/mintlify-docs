## Description

Cancel active quote subscriptions.

## Request - Command ID: 22006

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| cancel_type | uint32 | Yes | 0=cancel all, 1=cancel depth, 2=cancel trade, 3=cancel forex |

```json
{
  "cmd_id": 22006,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {
    "cancel_type": 1
  }
}
```

  "cmd_id": 22007,
  "seq_id": 123,
