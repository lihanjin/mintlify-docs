## Application Process

### Step 1 - Register

<Info>
Registration URL: [https://alltick.co/register](https://alltick.co/register)
</Info>

1. **Visit registration page**: Go to [https://alltick.co/register](https://alltick.co/register)
2. **Enter details**:
   - **Email**: Enter a valid email address
   - **Password**: Create a secure password (include uppercase, lowercase, special characters, and numbers)
   - **Verification code**: Enter the code sent to your email
3. **Complete registration**: Click "Register" and you'll be redirected to the dashboard

### Step 2 - Get API Token

1. **Enter dashboard**: After registration, navigate to the Dashboard
2. **Find Token**: Go to the "API keys" section
3. **Save Token**: Copy and securely store your Token

## Important Notes

- Keep your Token secure; do not share with untrusted third parties
- Contact support immediately if you encounter any issues during registration
- Always use the latest Token for API calls
