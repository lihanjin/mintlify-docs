This guide walks you through everything you need to make your first successful AllTick API call. By the end, you will have a working code example that fetches 10 daily K-line bars for AAPL.US using your own API token.

## 1. Register an account

Go to [alltick.co/register](https://alltick.co/register) and complete the sign-up form:

1. Enter a valid email address
2. Create a password (must include uppercase, lowercase, numbers, and special characters)
3. Enter the verification code sent to your inbox
4. Click Register — on success, you are redirected to your dashboard

## 2. Get your API token

In the dashboard, navigate to the **API Keys** section and copy your token.

<Warning>
Your token is your sole credential. If you regenerate it, the old token stops working immediately. Update all integrations.
</Warning>

## 3. Choose the right endpoint

AllTick uses separate base URLs by asset class:

| Asset class | Base URL |
|-------------|----------|
| US stocks, HK stocks, A-shares, indices | `https://quote.alltick.co/quote-stock-b-api` |
| Forex, crypto, precious metals, commodities | `https://quote.alltick.co/quote-b-api` |

## 4. Make your first request

Query the 10 most recent daily K-line bars for AAPL.US:

<Card title="Try it in the API Playground" icon="play" href="/en/api-reference/stock/kline">
  Open the K-line endpoint's interactive Playground — just fill in your token and hit send
</Card>

Code examples:

<CodeGroup>

```python Python
import requests
import json
import urllib.parse

token = "YOUR_TOKEN"
query = json.dumps({
    "trace": "test-001",
    "data": {
        "code": "AAPL.US",
        "kline_type": 8,
        "kline_timestamp_end": 0,
        "query_kline_num": 10,
        "adjust_type": 0
    }
})

url = f"https://quote.alltick.co/quote-stock-b-api/kline?token={token}&query={urllib.parse.quote(query)}"
response = requests.get(url)
print(json.dumps(response.json(), indent=2))
```

```go Go
package main

import (
    "fmt"
    "io"
    "net/http"
    "net/url"
)

func main() {
    token := "YOUR_TOKEN"
    query := `{"trace":"test-001","data":{"code":"AAPL.US","kline_type":8,"kline_timestamp_end":0,"query_kline_num":10,"adjust_type":0}}`

    reqURL := fmt.Sprintf("https://quote.alltick.co/quote-stock-b-api/kline?token=%s&query=%s",
        token, url.QueryEscape(query))

    resp, err := http.Get(reqURL)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()
    body, _ := io.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```

</CodeGroup>

Replace `YOUR_TOKEN` with the actual token from your dashboard.
