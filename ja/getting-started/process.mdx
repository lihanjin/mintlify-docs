このガイドでは、AllTick API の最初の呼び出しを成功させるまでの全手順を説明します。最終的に、自分の Token を使って AAPL.US の日足 K ラインデータを取得する動作コード例が得られます。

## 1. アカウント登録

[alltick.co/register](https://alltick.co/register) にアクセスして登録を完了します：

1. 有効なメールアドレスを入力
2. パスワードを設定（大文字、小文字、数字、特殊文字を含む）
3. メールに届いた認証コードを入力
4. 登録をクリック — 成功するとダッシュボードにリダイレクト

## 2. API Token を取得

ダッシュボードで **API Keys** セクションに移動し、Token をコピーします。

<Warning>
Token は唯一の認証情報です。再生成すると古い Token は即座に無効になります。すべての統合を更新してください。
</Warning>

## 3. エンドポイントを選択

AllTick は資産クラスごとに異なる Base URL を使用します：

| 資産クラス | Base URL |
|-----------|----------|
| 米国株、香港株、A 株、指数 | `https://quote.alltick.co/quote-stock-b-api` |
| 外国為替、暗号通貨、貴金属、コモディティ | `https://quote.alltick.co/quote-b-api` |

## 4. 最初のリクエストを実行

AAPL.US の直近 10 本の日足 K ラインを取得する例：

<Card title="API Playground で試す" icon="play" href="/ja/api-reference/stock/kline">
  K ラインエンドポイントのインタラクティブ Playground を開き、Token を入力してリクエストを送信
</Card>

コード例：

<CodeGroup>

```python Python
import requests
import json
import urllib.parse

token = "YOUR_TOKEN"
query = json.dumps({
    "trace": "test-001",
    "data": {
        "code": "AAPL.US",
        "kline_type": 8,
        "kline_timestamp_end": 0,
        "query_kline_num": 10,
        "adjust_type": 0
    }
})

url = f"https://quote.alltick.co/quote-stock-b-api/kline?token={token}&query={urllib.parse.quote(query)}"
response = requests.get(url)
print(json.dumps(response.json(), indent=2))
```

```go Go
package main

import (
    "fmt"
    "io"
    "net/http"
    "net/url"
)

func main() {
    token := "YOUR_TOKEN"
    query := `{"trace":"test-001","data":{"code":"AAPL.US","kline_type":8,"kline_timestamp_end":0,"query_kline_num":10,"adjust_type":0}}`

    reqURL := fmt.Sprintf("https://quote.alltick.co/quote-stock-b-api/kline?token=%s&query=%s",
        token, url.QueryEscape(query))

    resp, err := http.Get(reqURL)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()
    body, _ := io.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```

</CodeGroup>

`YOUR_TOKEN` をダッシュボードで取得した実際の Token に置き換えてください。
