## 説明

クライアントは10秒ごとにハートビートを送信する必要があります。30秒以内にハートビートが受信されない場合、サーバーはWebSocket接続を切断します。

## リクエスト - コマンドID：22000

```json
{
  "cmd_id": 22000,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```
