## 説明

アクティブな相場購読をキャンセルします。

## リクエスト - コマンドID：22006

| フィールド | 型 | 必須 | 説明 |
|-----------|------|------|------|
| cancel_type | uint32 | はい | 0=全てキャンセル、1=板情報キャンセル、2=約定キャンセル、3=為替キャンセル |

```json
{
  "cmd_id": 22006,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {
    "cancel_type": 1
  }
}
```
